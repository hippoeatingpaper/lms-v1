## 주의사항

1. **Vite 프록시**: 개발 환경에서만 동작, 프로덕션은 Nginx/같은 origin 필요
2. **쿠키**: SameSite 설정으로 인한 크로스 도메인 이슈 주의
3. **HMR**: 상태가 유지되지 않을 수 있음, 필요 시 persist 사용
4. **빌드**: 프로덕션 빌드 후 정적 파일 서빙 테스트 필수
